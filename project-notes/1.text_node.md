## 1.0 TextNode and Dunder
**Create a text node that  takes:**
- **text**
- **text_type**
- **url** *= None* (default) if images or anchor tag.

## 1.1 Create an equality dunder method and a repr for object representation:

```py
    def __eq__(self, other):
        # if all propertities from self and other obj match, return TRUE
        return (self.text == other.text) and (self.text_type == other.text_type) and (self.url == other.url)
```

#### Repr
```py
    def __repr__(self):
        return f"TextNode({ self.text}, {self.text_type}, {self.url})"
```
Just returns the print format when we do print(obj)
