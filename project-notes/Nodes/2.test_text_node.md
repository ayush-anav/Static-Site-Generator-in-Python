# Adding Tests to TextNode Class!
#### We will use unittest.

## Step 1: Create Test.sh script, add: 
```sh
    python3 -m unittest discover -s src
```

##### This command tells Python to:

- *run the unittest module* **(-m unittest)**
- use its **test *discovery* mode** *(discover)*
- look in the **src directory for tests (-s src)**

*btw, -s means start discovery from file path SRC*

--- 

##### By default, discovery will:

find files named **test*.py** **(e.g. test_textnode.py)**
**inside those files, run methods that start with test_ on subclasses of unittest.TestCase**
So: “use unittest, discover tests in src, using the test*.py naming convention.”

---

## Step 2: create test_textnode.py
#### INSIDE THE FILE, FEW MORE RULES:

- **Test class name** **should** use **subclass (inherit) unittest.TestCase**
- Any tests or any methods that we want to auto run inside the class, should be named like **test_METHOD_NAME**

```py
    def test_eq(self):
        ... 
```
---

**Stick to:**

**file:** test_*.py
**class:** class TestSomething(unittest.TestCase):
**methods:** def test_something(self):
and discover will happily find and run them.
