# HTMLNode
TextNode = inline 
### HTMLNode will represent a node in HTML document tree like:
```html
    <p>CONTENTS</p>
    <h1>
        <p> </p>
    </h1>
```
### AND is designed to only output HTML

---

### CONTRUCTOR WILL HAVE:
##### tag = HTML tag name e.g p, a, h1
##### value = what is inside the tag
##### children = list(HTMLNode objects)
##### props = {k, v} | k,v => attr, val
**AND THE attributes will be None by default, so optional.**

# Method 1:
#### NotImplementedError() means that the class will be overridden

- This **method is meant to be implemented by a subclass.**

- The current class is kind of an **“abstract” base: it defines the interface, not the full behavior.**

- If you call that method **without overriding it, Python raises NotImplementedError to say**:
“This method isn’t actually implemented here.”

```py
    def to_html(self):
        raise NotImplementedError()
```

# Method 2:
### Convert self.props if it exists into a attribute, value string that we will inject into our html element.

- take a string thats empty
- for k,v (attr, val) in self.props
- append it into the string and add space

FINAL OUTPUT:
href="https://www.google.com" target="_blank"

```py

    def props_to_html(self):
        if not self.props:
            return ""
        
        link = ""
        for attr, val in self.props.items():
            link += f' {attr}="{val}"'

        return link
```
